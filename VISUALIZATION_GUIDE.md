# 因子看板可视化指南

## 概述

基于对专业量化机构的研究，我们实现了一个**交互式 HTML 因子看板**，包含 6 大核心模块。

## 🎨 专业因子看板的 6 大核心组件

### 1️⃣ 核心指标展示 - KPI Cards

**作用**：一目了然地展示关键绩效指标

**包含指标**：
- 累计收益率
- 年化收益率
- 夏普比率
- 最大回撤
- 卡尔玛比率
- 胜率

**可视化方式**：
- 大号数字显示
- 颜色编码（绿涨红跌）
- 趋势箭头

---

### 2️⃣ 净值曲线

**作用**：展示策略的净值增长轨迹

**可视化方式**：
- 折线图或面积图
- 策略 vs 基准双线对比
- 零线参考线
- 交互式缩放

**关键要素**：
- 时间序列（X轴）
- 净值（Y轴）
- 颜色区分（策略=蓝色，基准=灰色）

---

### 3️⃣ 风险分析图表

#### 回撤图（水下图）
**作用**：可视化最大亏损深度和持续时间

**可视化方式**：
- 面积图（红色半透明填充）
- 负值显示（水下）
- 标注最大回撤点

#### 波动率
**作用**：展示收益率的波动程度

**可视化方式**：
- 滚动标准差时间序列
- 均值参考线

---

### 4️⃣ 因子有效性分析

#### IC 分布直方图
**作用**：评估因子预测能力的分布

**可视化方式**：
- 直方图 + 正态分布拟合
- 均值线标注
- 频数统计

#### IC 滚动分析
**作用**：监控因子稳定性的时间变化

**可视化方式**：
- 滚动窗口 IC 时间序列
- 均值 ± 1 标准差带

#### IC 衰减曲线
**作用**：评估因子预测能力的衰减

**可视化方式**：
- 多周期 IC 柱状图
- 衰减趋势线

---

### 5️⃣ 持仓分析

#### 换手率图
**作用**：评估交易频率和成本

**可视化方式**：
- 时间序列柱状图
- 平均换手率参考线
- 颜色编码（高换手=警示）

#### 行业分布
**作用**：分析行业暴露度

**可视化方式**：
- 饼图或环形图
- 百分比标签

#### 持仓集中度
**作用**：评估风险集中程度

**可视化方式**：
- 赫芬达尔指数时间序列
- 洛伦兹曲线

---

### 6️⃣ 对比分析

#### 基准对比
**作用**：策略 vs 基准表现对比

**可视化方式**：
- 双线对比图（净值曲线）
- 超额收益面积图
- 相对 strength 指标

#### 多因子雷达图
**作用**：多因子性能对比

**可视化方式**：
- 雷达图（6-8 个维度）
- 因子对比
- 颜色区分

## 🚀 使用方法

### 快速生成 HTML 报告

```python
from backtest import VectorBacktest, FactorReport

# 1. 运行回测
backtest = VectorBacktest(
    factor_data=factor_df,
    price_data=price_df,
    select_method="top_n",
    select_params={"top_n": 10},
)
result = backtest.run()

# 2. 生成 HTML 报告
report = FactorReport(result)
report.generate_html_report(
    output_path="reports/my_factor_report.html",
    title="MA20 因子回测报告",
)

# 3. 在浏览器中打开
# Windows: start reports/my_factor_report.html
# Mac: open reports/my_factor_report.html
```

### 自定义报告

```python
# 创建单个图表
kpi_fig = report.create_kpi_cards()
equity_fig = report.create_equity_curve()
drawdown_fig = report.create_drawdown_chart()
returns_dist_fig = report.create_returns_distribution()
monthly_heatmap_fig = report.create_monthly_returns_heatmap()
turnover_fig = report.create_turnover_chart()

# 单独保存为 HTML
kpi_fig.write_html("kpi_cards.html")
equity_fig.write_html("equity_curve.html")
```

## 🛠️ 技术栈

### 推荐的 Python 可视化库

| 库 | 优势 | 适用场景 |
|----|------|----------|
| **Plotly** | 交互式、HTML 友好 | 因子看板、报告生成 |
| **Matplotlib** | 静态图表、出版质量 | 学术论文、报告 |
| **Seaborn** | 统计图表、美观 | 数据探索、分析 |
| **Bokeh** | 实时数据、大数据 | 实时监控、大屏展示 |
| **Altair** | 声明式、简洁 | 快速原型、分析 |

### 我们的选择：Plotly

**原因**：
1. ✅ **交互性强**：缩放、平移、悬停
2. ✅ **HTML 输出**：直接生成独立 HTML 文件
3. ✅ **美观**：专业的金融图表样式
4. ✅ **易于使用**：简洁的 API
5. ✅ **开源免费**：无许可证限制

## 🔧 常见问题

### Q1: 如何自定义报告样式？
**A**: 修改 `generate_html_report()` 方法中的 CSS 样式。

### Q2: 如何添加自定义图表？
**A**: 继承 `FactorReport` 类，添加新的 `create_xxx_chart()` 方法。

### Q3: 如何集成到 Dash 应用？
**A**: Plotly 图表可以直接嵌入 Dash 应用。

### Q4: 如何导出为 PDF？
**A**: 浏览器打开 HTML 后，使用"打印"功能保存为 PDF。
